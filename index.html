About Me

I am a problem solver, but for now let's just go with programmer (proʊgræməʳ) and ui ux designer (dɪzaɪnəʳ)...Let's say, I bring programs and design concepts to life.

On the web: I leverage the power of React and Next.js to develop user-friendly, interactive websites. My experience with these frameworks allows me to build performant and scalable web experiences.

For mobile: I bring applications to life using React Native and Expo. This combination enables me to create native-looking mobile apps with a streamlined development process.

GCP: I leverage Kubernetes Engine, a managed Kubernetes service. Rust: Rust is a powerful and memory safe language that is suitable for resource-constrained devices and I leverage its power in the embedded field.
